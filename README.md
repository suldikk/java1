# Практическая работа №15

Java Swing приложение с формой регистрации пользователя.

## Возможности

- ввод имени;
- ввод email;
- ввод пароля;
- проверка заполнения всех полей;
- вывод результата регистрации.

## Запуск

```powershell
javac -encoding UTF-8 -d out src/Main.java
java -cp out Main
```
