# Практическая работа №15

Java Swing приложение с формой регистрации пользователя.

## Возможности

- ввод имени;
- ввод email;
- ввод пароля;
- проверка заполнения всех полей;
- проверка имени, email и пароля;
- сохранение зарегистрированных пользователей в локальные файлы `users.csv` и `users.txt`;
- вывод списка зарегистрированных пользователей в окне программы.

## Запуск

```powershell
javac -encoding UTF-8 -d out src/Main.java
java -cp out Main
```

Файлы `users.csv` и `users.txt` создаются автоматически после первой успешной регистрации.
Пароль в файлы не сохраняется. Он используется только для проверки формы.
